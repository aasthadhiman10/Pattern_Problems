# Pattern-Problems

This repository is dedicated to mastering **Nested Loops** and **Algorithmic Pattern Generation** in Java. Pattern problems are a vital exercise for strengthening logical thinking, understanding row-column coordination, and managing complex iterative control in code.

## 🚀 Projects Included

* **Problem_1.java:** Solid Rectangle implementation using basic nested loop control.
* **Problem_2.java:** Hollow Rectangle pattern with conditional boundary logic.
* **Problem_3.java:** Half Pyramid pattern demonstrating row-dependent column growth.
* **Problem_4.java:** Inverted Half Pyramid using decrementing loop iteration.
* **Problem_5.java:** Right-Aligned (180° Rotated) Pyramid with space alignment logic.
* **Problem_6.java:** Half Pyramid with numbers (sequential sequence).
* **Problem_7.java:** Inverted Number Pyramid using reverse iteration.        
* **Problem_8.java:** 0-1 Triangle pattern using parity (even/odd) math.        


---

## 🛠️ Concepts Covered

* **Nested Loops:** Managing the interplay between outer row (`i`) and inner column (`j`) loops.
* **Conditional Logic:** Using `if-else` blocks for boundary checks and pattern shapes.
* **Mathematical Patterns:** Applying modulo operators (`%`) and variable sums for sequence generation.
* **Coordinate Management:** Manipulating print statements to achieve specific visual outputs.
